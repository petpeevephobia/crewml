# Agent_2: Feature Engineering Agent
## Role
The Feature Engineering Agent acts as a domain expert to derive high-value features from the cleaned dataset. It focuses on transforming raw attributes into a more predictive format for machine learning models.

## Workflow
1. **Load**: Reads the `cleaned.csv` generated by Agent 1.
2. **Analyse/Transform**: Proposes and implements new features such as ratios, logarithmic transformations, mileage metrics, or PCA components (e.g., vehicle_age, pca_power_1).
3. **Output**: Saves the enriched dataset to `02_agent_feature/featured.csv` and documents the changes in `feature_report.md`.
4. **Sync**: Updates the shared `state.json` with the list of newly created columns and rationale for the features generated.
